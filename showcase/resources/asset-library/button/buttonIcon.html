Buttons With Icons

To display an icon inside a button, simply add the icon inside the button and wrap the text in an inline element as shown in the HTML example below.

All modifiers for Button and Icon can be applied. The size modifiers for button and icon are designed to be used together.

To show a large button, apply the modifier ebBtn_large to the button, and apply the modifier ebIcon_large to the icon. A list of available icons can be seen in Icons section of the library.

HTML

<button class="ebBtn"><i class="ebIcon ebIcon_settings"></i> <span>Settings</span></button>
