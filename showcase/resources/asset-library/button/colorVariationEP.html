Color Variation: Ericsson Palette

The colour buttons use a combination of classes: eBtn and ebBtn_color_XXX. The appropriate modifiers for each button colour are listed below.

ebBtn_color_darkGreen
ebBtn_color_paleBlue
ebBtn_color_green
ebBtn_color_orange
ebBtn_color_red
ebBtn_color_purple

HTML

<button class="ebBtn ebBtn_color_darkGreen">Dark Green</button>
