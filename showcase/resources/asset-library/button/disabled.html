Disabled State

The disabled state uses the ebBtn_disabled class or the disabled attribute. This can be applied to any size of the standard button.

HTML

<button class="ebBtn" disabled>Lorem Ipsum</button>

Implementation Guideline

Another way to set a button to disabled state is available.
<button class="ebBtn" disabled="disabled">Lorem Ipsum</button>

If you are using JSCore, the button exists as Widget in the Widget Library #button
