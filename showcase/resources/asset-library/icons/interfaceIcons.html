Interface Icons

A selection of different icons are available in the Asset Library. All icons are supplied in Scalable Vector Graphics (.svg) format. The example code in this section applies the icon classes to <i> tags, however <span> or <div> tags would work equally well. The standard icon size is 16 x 16 px.
