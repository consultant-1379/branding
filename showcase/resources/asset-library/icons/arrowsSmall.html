Arrow Small Icons

Implementation Guideline

Small arrow icons are placed into small box of 10x10px. Standard arrow icons (16x16px) modification is not working correctly with the "small" modifier. In this case you should add modifier value "10px" for icon name.
  • ebIcon_upArrow_10px
  • ebIcon_downArrow_10px
  • ebIcon_leftArrow_10px
  • ebIcon_rightArrow_10px
  • ebIcon_prevArrow_10px
  • ebIcon_nextArrow_10px

HTML

<i class="ebIcon ebIcon_small ebIcon_upArrow_10px"></i>
