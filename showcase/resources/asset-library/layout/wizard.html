Wizard can be used with the following set of modifiers:

ebWizard
ebWizard_horizontal Renders the step list horizontally on a screen sized device
ebWizard_firstStep Hides the "previous" button
ebWizard_lastStep Hides the "next" button and show "finish"


It is possible to disable navigation button on the header using the following modifiers:

ebWizard-headerStepSelectPrevious,
ebWizard-headerStepSelectNext
ebWizard-headerStepSelectPrevious_disabled
ebWizard-headerStepSelectNext_disabled 		Shows the corresponding header button disabled
ebWizard-headerStepSelectPrevious_hidden
ebWizard-headerStepSelectNext_hidden 		Hides the corresponding header button without reducing width


The step item header can be set to valid, disabled or highlighted as current using the following modifiers:

ebWizard-step
ebWizard-step_current Shows corresponding step as current
ebWizard-step_valid Shows corresponding step as valid
ebWizard-step_disabled Shows corresponding step as disabled
