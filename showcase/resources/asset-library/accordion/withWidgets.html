Accordion with widgets

The accordion can content widgets in both areas. If title area is not clickable, add the modifier "noAction" for header class name "ebAccordion-header".



HTML

<div class="ebAccordion">
  <div class="ebAccordion-header ebAccordion-header_noAction">
    <div class="ebAccordion-button">
      <i class="ebIcon ebIcon_small ebIcon_upArrow_10px"></i>
    </div>
    <div class="ebAccordion-title">
      <label>
        <input class="ebCheckbox" type="checkbox" value="all"/>
        <span class="ebCheckbox-inputStatus"></span>
        <span class="ebCheckbox-label">Checkboxes</span>
      </label>
    </div>
  </div>
  <div class="ebAccordion-body">
    <ul class="ebList">
      <li class="ebList-item">
        <label>
          <input class="ebCheckbox" type="checkbox" value="1"/>
          <span class="ebCheckbox-inputStatus"></span>
          <span class="ebCheckbox-label">Checkbox 1</span>
        </label>
      </li>
      <li class="ebList-item">
        <label>
          <input class="ebCheckbox" type="checkbox" value="2"/>
          <span class="ebCheckbox-inputStatus"></span>
          <span class="ebCheckbox-label">Checkbox 2</span>
        </label>
      </li>
    </ul>
  </div>
</div>
