Common class names

Common class names are globally defined and can be used as part of your element.


eb_wMargin    Adds "margin" style for the element
eb_noVertAlign    Removes "vertical-align" style from the element


HTML

<button class="ebBtn">Simple button</button>
<button class="ebBtn eb_wMargin">eb_wMargin</button>
<button class="ebBtn eb_noVertAlign">eb_noVertAlign</button>
