Notification with icon

You can include an icon inside the notification for better recognition of the notification type. You can use any of the icons available from the branding assets.
Error Message

HTML

<div class="ebNotification ebNotification_color_red">
  <div class="ebNotification-content">
    <span class="ebNotification-icon">
      <i class="ebIcon ebIcon_error"></i>
    </span>
    <span class="ebNotification-label">Error Message</span>
    <span class="ebNotification-close">
      <i class="ebIcon ebIcon_close_red ebIcon_interactive"></i>
    </span>
  </div>
</div>

Implementation Guideline

Ensure that your icon matches the notification colour.
