Notification Colours

The colour notifications use a combination of classes: ebNotification and ebNotification_color_XXX. The appropriate modifiers for each notification colour are listed below.
Info Message
ebNotification_color_paleBlue
OK Message
ebNotification_color_green
Warning Message
ebNotification_color_yellow
Error Message
ebNotification_color_red

Implementation Guideline

Ensure to match the colours of the notification and the close button.
