Toast Notification

You can use notification as a toast notification by adding ebNotification_toast to the root element of the notification. This will cause the notification to appear at the top of its parent, centered horizontally.
Parent div
Default Message

HTML

<div class="ebNotification ebNotification_toast">
  <div class="ebNotification-content">
    <span class="ebNotification-label">Default Message</span>
    <span class="ebNotification-close">
      <i class="ebIcon ebIcon_close ebIcon_interactive"></i>
    </span>
  </div>
</div>

Implementation Guideline

Add the toast notification to the application's root element to have application level notifications.
