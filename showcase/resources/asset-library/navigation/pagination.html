Pagination

A "UL" list is used to create the page numbering. The modifier "ebPagination-entryAnchor_current" is used to set the currently selected page. The modifier "ebPagination-entryAnchor_dots" is used to disable pointer events on the "..." anchor elements.

HTML

<ul class="ebPagination">
  <li class="ebPagination-previous">
    <a class="ebPagination-previousAnchor" href="#"></a>
  </li>
  <li class="ebPagination-pagesContainer">
    <ul class="ebPagination ebPagination-pages">
      <li class="ebPagination-entry">
        <a class="ebPagination-entryAnchor" href="#">1</a>
      </li>
      <li class="ebPagination-entry">
        <a class="ebPagination-entryAnchor ebPagination-entryAnchor" href="#">2</a>
      </li>
      <li class="ebPagination-entry">
        <a class="ebPagination-entryAnchor" href="#">3</a>
      </li>
      <li class="ebPagination-entry">
        <a class="ebPagination-entryAnchor ebPagination-entryAnchor_current" href="#">4</a>
      </li>
      <li class="ebPagination-entry">
        <a class="ebPagination-entryAnchor" href="#">5</a>
      </li>
    </ul>
  </li>
  <li class="ebPagination-next">
    <a class="ebPagination-nextAnchor" href="#"></a>
  </li>
</ul>
