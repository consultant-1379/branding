Text Modifiers

These text modifiers can be applied to any HTML element which is used to display body text, e.g. <p>, <div>, <span> etc.

Small text

<p class="ebText_small">Small text</p>

Alternative text

<p class="ebText_alternative">Alternative text</p>

Disabled text

<p class="ebText_disabled">Disabled text</p>

Error text

<p class="ebText_error">Error text</p>
