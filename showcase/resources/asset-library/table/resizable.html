Resizable Table

If you need a column resizable table, insert the ebTable-headerResize element inside the header and set the class "ebTable-th_resizable" on the corresponding th.

Implementation Guideline

You need to add the class "ebTable-th_resizable" to the table headers row th that are resizable.
