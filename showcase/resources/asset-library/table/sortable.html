Sortable Table

If you need a column sortable table, insert the ebSort inside the header.

Header 1
Header 2
Header 3

HTML

<table class="ebTable">
<thead>
<tr>
  <th class="ebTable-th_sortable">
    <div class="ebTable-header">
      <span class="ebTable-headerText">Header 1</span>
      <span class="ebTable-headerSort ebSort">
        <i class="ebSort-arrow_up ebSort-arrow_active"></i>
        <i class="ebSort-arrow_down"></i>
      </span>
    </div>
  </th>
  <th class="ebTable-th_sortable" >
    <div class="ebTable-header">
      <span class="ebTable-headerText">Header 2 long line</span>
      <span class="ebTable-headerSort ebSort">
        <i class="ebSort-arrow_up"></i>
        <i class="ebSort-arrow_down"></i>
      </span>
    </div>
  </th>
  <th class="ebTable-th_sortable">
    <div class="ebTable-header">
      <span class="ebTable-headerText">Header 3 wrapped line</span>
      <span class="ebTable-headerSort ebSort">
        <i class="ebSort-arrow_up"></i>
        <i class="ebSort-arrow_down"></i>
      </span>
    </div>
  </th>
</tr>
</thead>
<tr>
  <td>Entry 1</td>

  <td>Entry 3</td>
</tr>

  <td>Entry 12</td>

Implementation Guideline

You need to add the class "ebTable-th_sortable" to the table headers row th that are sortable.
