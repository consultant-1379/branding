Info Popup Variations

Four variations of the Info Popup are available, each with the popup in a different position relative to the info icon. The classes that are used to determine which version is shown are:

ebInfoPopup-content_corner_topRight
ebInfoPopup-content_corner_bottomLeft
ebInfoPopup-content_corner_bottomRight
