Dialog With Optional Block

The dialog box with optional block could be used when the user is asked to make an action to continue his work. It could be confirmation or warning dialog box.
Primary text

Secondary text providing further details.

HTML

<div class="ebDialogBox">
  <div class="ebDialogBox-contentBlock">
    <div class="ebDialogBox-primaryText">Primary text</div>
    <div class="ebDialogBox-secondaryText ebText_alternative">
      <p>Secondary text providing further details.</p>
    </div>
    <div class="ebDialogBox-thirdText">
      <label>
        <input class="ebCheckbox" type="checkbox" value="1"/>
        <span class="ebCheckbox-inputStatus"></span>
        <span class="ebCheckbox-label">Option</span>
      </label>
    </div>
  </div>
  <div class="ebDialogBox-actionBlock">
    <button class="ebBtn ebBtn_color_darkBlue">OK</button>
    <button class="ebBtn">Cancel</button>
  </div>
</div>
