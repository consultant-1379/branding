Dialog With Optional Close X

The dialog box can have optional close button in the top right corner. This button should close the dialog box.
Primary text

Secondary text providing further details. Also include information that explains any possible consequences of the affirmative action or alternate actions.

HTML

<div class="ebDialogBox">
  <div class="ebDialogBox-contentBlock">
    <div class="ebDialogBox-icon">
      <span class="ebIcon ebIcon_close ebIcon_interactive"></span>
    </div>
    <div class="ebDialogBox-primaryText">Primary text</div>
    <div class="ebDialogBox-secondaryText ebText_alternative">
      <p>Secondary text providing further details. [...]</p>
    </div>
  </div>
  <div class="ebDialogBox-actionBlock">
    <button class="ebBtn ebBtn_color_darkBlue">OK</button>
    <button class="ebBtn">Cancel</button>
  </div>
</div>
