Dialog With One Button

Dialog boxes are used to help the user complete an action. Dialogs can contain text and various types of input, such as settings or property editing functionality.
The dialog box with one button is used when the user is informed about some kind of information. It could be information or error dialog box.
Primary text

Secondary text providing further details.

HTML

<div class="ebDialogBox">
  <div class="ebDialogBox-contentBlock">
    <div class="ebDialogBox-primaryText">Primary text</div>
    <div class="ebDialogBox-secondaryText">
      <p>Secondary text providing further details.</p>
    </div>
  </div>
  <div class="ebDialogBox-actionBlock">
    <button class="ebBtn ebBtn_color_darkBlue">Close</button>
  </div>
</div>
