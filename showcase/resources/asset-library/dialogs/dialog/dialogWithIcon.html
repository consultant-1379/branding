Dialog With Icon

The dialog box can have icon in the left top corner. This icon help better to introduce type of information.
There are four main types of icons:
Information icon is used for Information Message Dialog boxes.
Warning icon is used for Warning Dialog boxes.
Confirmation icon is used for Question/Confirmation Dialog boxes.
Error icon is used for Error Dialog boxes.


To change icon the "type" modifier for "ebDialogBox-contentBlock" should be used. There are four defined values for this modifier: "information", "warning", "confirmation" and "error".
