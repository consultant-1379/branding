Dialog With Red Button

The Dialog box with red button is used when a primary action is dangerous or destructive.
System Shutdown

Action will shutdown the entire system!

Are you sure you want to proceed?

<div class="ebDialogBox">
  <div class="ebDialogBox-contentBlock ebDialogBox-contentBlock_type_warning">

    <div class="ebDialogBox-secondaryText ebText_alternative">
      <p>Action will shutdown the entire system!</p>
    </div>
    <div class="ebDialogBox-thirdText">
      <p>Are you sure you want to proceed?</p>
    </div>
  </div>
  <div class="ebDialogBox-actionBlock">
    <button class="ebBtn ebBtn_color_red">Shutdown</button>
    <button class="ebBtn">Don't Shutdown</button>
  </div>
</div>
