Checkboxes

Checkboxes are for selecting one or several options in a list or table. To use the checkbox asset an extra <span> tag is required to show the input status (see HTML example below).

HTML

<label>
  <input class="ebCheckbox" type="checkbox" value="1"/>
  <span class="ebCheckbox-inputStatus"></span>
  <span class="ebCheckbox-label">Label</span>
</label>
