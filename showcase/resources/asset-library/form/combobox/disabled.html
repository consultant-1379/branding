Disabled State

The disabled state must have the modifier "ebCombobox_disabled" as well as the button and input disabled using the attribute disabled.

  • 2

HTML

<div class="ebCombobox ebCombobox_disabled">
  <input type="text" class="ebInput ebInput_noRightSide" disabled/>
  <button class="ebCombobox-helper" disabled>
    <span class="ebCombobox-iconHolder">
      <i class="ebIcon ebIcon_small ebIcon_downArrow_10px eb_noVertAlign"></i>
    </span>
  </button>
  <div class="ebCombobox-body">
    <ul class="ebComponentList">
      <li class="ebComponentList-item">1</li>
      <li class="ebComponentList-item">2</li>
    </ul>
  </div>
</div>
