Label modifier

Labels can be set on the the input using the modifier ebInput_labeled and enclosing the input tag in a label tag. The label appears on the left by default.
<label class="ebLabel">
  Label
  <input type="text" class="ebInput ebInput_labeled eb_wMargin"
    placeholder="Input"/>
</label>
To show a label on the top of the input field, use the modifier ebInput_labeled_top instead of ebInput_labeled.

Removing Borders

The modifiers "ebInput_noLeftSide" and "ebInput_noRightSide" can be used to remove the left or right borders from the input field.
