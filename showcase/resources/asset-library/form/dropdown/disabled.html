Disabled State

The disabled state must have the modifier ebDropdown_disabled as well as the button disabled using the attribute disabled.

  • Option 2

    <span class="ebDropdown-iconHolder">

    </span>
  </button>
  <div class="ebDropdown-body">
    <ul class="ebComponentList">
      <li class="ebComponentList-item">Option 1</li>
      <li class="ebComponentList-item">Option 2</li>
    </ul>
  </div>
</div>
