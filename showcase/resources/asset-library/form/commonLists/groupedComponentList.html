Grouped component List

You can group your items using "ebComponentList-group" as follow
  • Group 1
    • Options one long
    • Option 2 text long long
  • Group 2 long name
    • Options one link long long long long
    • Option 2 shorter





HTML

<ul class="ebComponentList">
  <li class="ebComponentList-group">
    <div class="ebComponentList-group-header">Group 1</div>
    <ul class="ebComponentList-inner">
      <li class="ebComponentList-item">
        <a class="ebComponentList-link" href="">Options one long</a></li>
      <li class="ebComponentList-item">Option 2 text long long</li>
    </ul>
  </li>
  <li class="ebComponentList-group">
    <div class="ebComponentList-group-header">Group 2 long name</div>
    <ul class="ebComponentList-inner">
      <li class="ebComponentList-item">
        <a class="ebComponentList-link" href="">Options one link long long</a></li>
      <li class="ebComponentList-item ebComponentList-item_selected">
        <a class="ebComponentList-link" href="">Option 2 shorter</a>
      </li>
    </ul>
  </li>
</ul>
