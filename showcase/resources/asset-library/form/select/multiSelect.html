Multi Select Box

The multi select presents a list of options and allows the user to select some of them. The multi select label is replaced with the info how many items are selected. If items are not selected, than the label has default text "Select options".




HTML

<div class="ebSelect">
  <button class="ebSelect-header">
    <span class="ebSelect-value" title="Options">Options</span>
    <span class="ebSelect-iconHolder">
      <i class="ebIcon ebIcon_small ebIcon_downArrow_10px"></i>
    </span>
  </button>
  <div class="ebSelect-body">
    <ul class="ebComponentList">
      <li class="ebComponentList-item" title="Value 1">
        <label>
          <input class="ebCheckbox" type="checkbox" value="Value1" checked>
          <span class="ebCheckbox-inputStatus"></span>
          <span class="ebCheckbox-label">Value 1</span>
        </label>
      </li>
      <li class="ebComponentList-item" title="Value 2">
        <label>
          <input class="ebCheckbox" type="checkbox" value="Value2" checked>
          <span class="ebCheckbox-inputStatus"></span>
          <span class="ebCheckbox-label">Value 2</span>
        </label>
      </li>
      <li class="ebComponentList-item" title="Value 3">
        <label>
          <input class="ebCheckbox" type="checkbox" value="Value3">
          <span class="ebCheckbox-inputStatus"></span>
          <span class="ebCheckbox-label">Value 3</span>
        </label>
      </li>
    </ul>
  </div>
</div>
