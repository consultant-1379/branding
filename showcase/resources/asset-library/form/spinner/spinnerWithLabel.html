Spinner Labels

A label can be added to the left or right side of the Spinner by using the "ebSpinner-label" or "ebSpinner-value" class name along with the corresponding markup.
Label
Value

HTML

<table class="ebSpinner">
  <tr>
    <td class="ebSpinner-label">Label</td>
    <td class="ebSpinner-holder">
      <div class="ebSpinner-iconUpHolder">
        <i class="ebIcon ebIcon_upArrow ebIcon_interactive ebIcon_noVertAlign"></i>
      </div>
      <div class="ebSpinner-inputHolder">
        <input type="text" class="ebInput ebInput_miniW ebInput_txtCenter" value="20"/>
      </div>
      <div class="ebSpinner-iconDownHolder">
        <i class="ebIcon ebIcon_downArrow ebIcon_interactive ebIcon_noVertAlign"></i>
      </div>
    </td>
    <td class="ebSpinner-value">Value</td>
  </tr>
</table>
