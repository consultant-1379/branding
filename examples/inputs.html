Input Validation

There are 3 types of messages

  • (?):   Info message
  • (ok): OK message
  • (X):   Error message

There are 4 different scenario:

  1. (X):       (default) The input does not show any message unless the input is wrong
  2. (? or X):     The input shows info message unless the input is wrong then display error
  3. (OK or X):    The input shows ok or error only
  4. (? then OK or X): The input shows info first then ok or error
